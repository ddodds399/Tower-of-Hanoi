40103628 - Towers of Hanoi Project
'Release Version'

Desktop Shortcut Included


Objective: Move all the rings to the third stand.
Below are the input commands:
Use the 'a' & 'd' keys to orbit the scene.
Use the 'f' key to toggle fullscreen mode.
Use the 'r' key to restart the game.
Use the left & right arrow keys to move the cursor over the towers.
Use the space bar to pickup and drop a ring.
Use the 'esc' key to exit the program.
